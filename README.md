# Nemodas (Network Monitoring Dashboard System)

Nemodas is a web-based network monitoring dashboard system that allows you to monitor your network devices in real-time.

## Features

- Real-time monitoring of device traffic
- Interactive graphs for traffic analysis
- Multi-device management
- Session logging
- Monthly reports

## Tech Stack

- Laravel
- Tailwind CSS
- Alpine.js
- Chart.js

## Installation

1. Clone the repository
2. Copy `.env.example` to `.env` and configure your database
3. Run `composer install`
4. Run `npm install`
5. Run `php artisan key:generate`
6. Run `php artisan migrate`
7. Run `npm run dev` and `php artisan serve`
